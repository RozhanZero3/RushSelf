RushSelf — سامانه سفارش غذای سلف دانشگاه
=========================================

معرفی
------
RushSelf یک سامانه ساده و سبک برای مدیریت منوی سلف دانشگاه و ثبت سفارش توسط دانشجویان است. این پروژه با Django ساخته شده و امکانات زیر را ارائه می‌دهد:

- مدیریت منوی روزانه به همراه آیتم‌های منو
- ثبت سفارش توسط دانشجویان با انتخاب غذا، مخلفات، و تعداد
- تولید کد رهگیری عددی کوتاه (۵-۱۲ رقم؛ در صورت تکرار طول افزایش می‌یابد)
- پشتیبانی از تاریخ شمسی (جلالی) در نمایش‌ها
- صفحه انتخاب نوع پرداخت (آنلاین / حضوری)
- پنل مدیریت با ظاهر بهبود یافته و راست‌چین
- قالب‌های Responsive و طراحی کارت‌محور برای منو

فناوری‌ها
--------
- Python 3.12+
- Django 6.x
- jdatetime (برای تبدیل تاریخ شمسی)
- SQLite (پیش‌فرض برای توسعه)

شروع سریع (محلی)
-----------------
در ویندوز (PowerShell) مراحل زیر را اجرا کنید:

```powershell
# ایجاد virtualenv (اختیاری اما پیشنهاد شده)
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# نصب وابستگی‌ها
pip install django jdatetime

# اعمال مایگریشن‌ها
python manage.py migrate

# (اختیاری) ایجاد یک superuser برای دسترسی به پنل مدیریت
python manage.py createsuperuser

# اجرای سرور توسعه
python manage.py runserver
```

پس از اجرای سرور به آدرس http://localhost:8000/ مراجعه کنید.

ساختار پروژه
-------------
- `canteen/` — اپلیکیشن اصلی شامل مدل‌ها، ویوها، قالب‌ها و فایل‌های استاتیک
- `rushself_project/` — پیکربندی پروژه Django (settings, urls, wsgi)
- `db.sqlite3` — دیتابیس SQLite محلی

توضیحات پیاده‌سازی
-------------------
- نمایش تاریخ‌ها در قالب‌ها و پنل مدیریت با تقویم شمسی انجام شده است. از `templatetags/shamsi.py` برای فیلترهای مرتبط استفاده شده است.
- مقادیر قیمت‌ها با جداکننده هزارگان نمایش داده می‌شوند و واحد `تومان` اضافه شده است. فیلتر مربوطه در `templatetags/currency.py` قرار دارد.
- برای انتخاب مخلفات به ازای هر آیتم از مدل `OrderItem` استفاده شده است که مخلفات هر آیتم را ذخیره می‌کند.
- پنل مدیریت با CSS سفارشی راست‌چین و رنگ‌بندی بنفش یکپارچه شده است؛ فایل CSS در `canteen/static/canteen/admin-custom.css` قرار دارد.

نکات توسعه و گسترش
------------------
- درگاه پرداخت آنلاین: در `templates/canteen/payment_result.html` یک لینک نمونه قرار داده شده است؛ برای اتصال به درگاه بایستی کد مربوط به پرداخت (API و callback) اضافه گردد.
- احراز هویت/مجوزها: در صورت نیاز به کنترل دقیق‌تر دسترسی مدیران و کاربران، قوانین و middleware مناسب اضافه شود.
- استقرار: برای محیط تولید از یک سرور WSGI/ASGI (مثل Gunicorn یا Daphne) به همراه یک وب‌سرور معکوس (Nginx) استفاده کنید و `DEBUG=False` قرار دهید.

چگونه کمک کنید
----------------
اگر می‌خواهید من:
- یک درگاه پرداخت مشخص (مثلا زرین‌پال، به‌پرداخت، یا درگاه دلخواه شما) را متصل کنم — نام و مدارک درگاه را بدهید.
- نمونه داده برای منو و مخلفات اضافه کنم تا تست UI سریع‌تری داشته باشید — بگو تا بنویسم و اجرا کنم.
- طراحی را بیشتر شخصی‌سازی کنم (لوگو، فونت، رنگ‌های دیگری) — رنگ‌ها و لوگو را بفرستید.
